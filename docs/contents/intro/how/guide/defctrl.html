Controls


For a simple Tigerian's application you can declare your controls and organise those on your web page like this: 
let myText = new Label(myApp, "Text 1");
let myContainer = new Container(myApp);
let myButton = new Button(myContainer, "Button 1");


Notice: You have to import Application and Controls that you want to use in your application at the top of your script. 
import { Label } from "tigerianjs/controls/Label.js"
