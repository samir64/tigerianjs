Use minified script

Use npm to install the latest released version:
npm install tigerian.js
 Copy two minified files ( 
tigerian.min.js
and 
tigerian.min.css
) from 
node_modules/tigerian.js
folder to your project folder.
After that you must add 
script
and 
style
tag in your HTML, between of 
head
tag for these script and stylesheet files.
